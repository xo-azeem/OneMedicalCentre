## USAGE GUIDELINES
===================

### Primary Brand Usage
- Use Yellow-400 (#FACC15) for main CTAs and primary actions
- Use Yellow-300 (#FDE047) for hover states and highlights
- Use Yellow-500 (#EAB308) for active states and pressed buttons

### Background Usage
- Use Gray-900 (#111827) for main page backgrounds
- Use Gray-800 (#1F2937) for card backgrounds and sections
- Use Gray-700 (#374151) for elevated elements

### Text Usage
- Use White (#FFFFFF) for primary text on dark backgrounds
- Use Gray-300 (#D1D5DB) for secondary text
- Use Yellow-400 (#FACC15) for accent text and links

### Border Usage
- Use Gray-700/50 for subtle borders
- Use Yellow-400/30 for accent borders
- Use White/20 for glass-morphism effects

## ACCESSIBILITY NOTES
=====================

### Contrast Ratios
- White text on Gray-900 background: High contrast (AAA)
- Yellow-400 text on Gray-900 background: Good contrast (AA)
- Gray-300 text on Gray-900 background: Good contrast (AA)

### Color Blindness Considerations
- Yellow and gray combination is accessible for most color vision deficiencies
- Always pair color with text or icons for important information
- Use sufficient contrast ratios for all text combinations
