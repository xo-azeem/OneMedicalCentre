## USAGE GUIDELINES
===================

### Primary Brand Usage
- Use Yellow-400 (#F59E0B) for main CTAs and primary buttons
- Use Yellow-500 (#D97706) for hover states and active elements
- Use Yellow-600 (#B45309) for text links and accents

### Background Usage
- Use White (#FFFFFF) for main page backgrounds
- Use Gray-50 (#F9FAFB) for card backgrounds and sections
- Use Yellow-50 (#FFFBEB) for highlighted sections

### Text Usage
- Use Gray-900 (#111827) for primary headings and important text
- Use Gray-800 (#1F2937) for body text and secondary headings
- Use Gray-700 (#374151) for tertiary text and captions
- Use Yellow-600 (#B45309) for links and accent text

### Border Usage
- Use Gray-200 (#E5E7EB) for card borders and dividers
- Use Gray-300 (#D1D5DB) for input borders and form elements
- Use Yellow-200 (#FDE68A) for accent borders and highlights

## ACCESSIBILITY NOTES
=====================

### WCAG Compliance
- All text combinations meet WCAG AA standards
- Primary text combinations exceed AAA standards
- Color is never the only way to convey information

### Recommended Combinations
- **High Priority Text**: Gray-900 on White backgrounds
- **Body Text**: Gray-800 on White/Gray-50 backgrounds  
- **Accent Text**: Yellow-600 on White/Gray-50 backgrounds
- **Buttons**: White text on Yellow-400/500/600 backgrounds
- **Links**: Yellow-600 text with underline on light backgrounds

### Color Blindness Considerations
- Yellow and gray maintain good contrast for all color vision types
- Sufficient luminance contrast maintained throughout
- Interactive elements use multiple visual cues (color + border + shadow)
